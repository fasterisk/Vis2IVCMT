How to use this program

Comparison Overview

On the left side, the upper view is the Comparison Overview. It contains one row, and one column per tree, so that you get a quick overview of the measurement between the trees.
Within the Comparison Overview, you can select the reference tree by clicking on a column. If you select a reference tree, all measures are recalculated and the views are updated.
